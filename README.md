# 🌐 Language Translation Tool

A simple and interactive Language Translation Tool built with Streamlit and Deep Translator API.

## Features
- Translate text into 10+ languages
- Auto language detection
- Clean and simple UI
- Copy translated text easily

## Languages Supported
English, Urdu, Arabic, French, Spanish, German, Chinese, Hindi, Turkish, Russian

## Installation

pip install -r requirements.txt

## Run the App

streamlit run app.py

## Built With
- Python
- Streamlit
- Deep Translator (Google Translate API)
